# 基本情報技術者試験 学習コマンド

questions/ディレクトリにある問題データを使って学習を行います。

## 使い方

ユーザーが `/study` を実行したら：

1. questions/sample.json から問題をランダムに1問選ぶ
2. 問題文と選択肢を表示
3. ユーザーの回答を待つ
4. 正誤判定と解説を表示
5. 続けるか聞く

## 実行手順

1. まず questions/sample.json を読み込む
2. ランダムに1問選んで出題
3. ユーザーが ア/イ/ウ/エ で回答
4. 正解なら「正解！」、不正解なら「残念！正解は○」と表示
5. 解説を表示
6. 「続ける？」と聞いて、続けるなら次の問題へ

## 出題形式

**【{category1} - {category3}】**
**{title}**

{question}

回答は？（ア/イ/ウ/エ）
