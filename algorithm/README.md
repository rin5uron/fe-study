# 🧮 アルゴリズム学習エリア

## 📚 学習ファイル構成

- **[algorithm_patterns.md](./algorithm_patterns.md)** - 基本情報技術者試験に頻出のアルゴリズムパターンBEST10

### 学んだことや気付き
- **[study](./study)** - 学習記録・理論・実践を記録

<br><br>

## 🎓 学習の進め方（Issue活用版）

1. **Issue作成** → githubCLIで`practice_problems.md`から問題を選択
2. **解答をコメント** → 考え方も含めて記載
3. **AIが解説・フィードバック** → Githubactionsで正誤判断と詳しい解説を自動生成
